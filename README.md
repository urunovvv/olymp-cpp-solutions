# Olymp C++ Solutions

My competitive programming solutions in **C++** (with some Python) from **olymp.tj** and various other platforms and contests.

## About

This repository contains my accepted solutions to algorithmic and olympiad programming problems.  
The main focus is on problems from the **Tajikistani Olympiad platform (olymp.tj)**, but it also includes solutions from:

- Codeforces
- ACMP (acmp.ru)
- Foxford
- Electicode
- Informatics.com
- Various training contests and seasonal contests (Good Bye 2025, Hello 2026!, MOM-2025, etc.)

## Repository Structure

The solutions are organized into folders by platform, contest, or topic:

- **`olymp.tj`** — Main collection from olymp.tj
- **`CodeForces`** — Solutions from Codeforces contests and problems
- **`acmp`**  — Solutions from acmp.ru
- **`foxford`** — Foxford problems
- **`AlgoMath`** — Mathematical algorithms and problems
- **`graph`** / **`Informatics.com/Graph`** — Graph theory problems
- **`Algo Data-Guide`** — Data structures and algorithms guide / implementations
- **`Hometasks`** — Homework and training tasks
- **`training_contests`** — Various training rounds
- Contest-specific folders: `Good Bye 2025`, `Hello, 2026!`, `mom-2025`, `mind_palace_marathon`, `Snowflake Strings`, etc.

Most files are `.cpp`. A small portion is in Python.

## How to Use

- Browse folders to find solutions by contest or topic.
- Each solution is usually a standalone C++ file with the problem code or name.
- Feel free to use these as reference for learning or practice.

**Note:** These are my personal solutions. There may be cleaner or more optimal approaches — always try to understand the logic rather than just copying.

## Technologies

- **Primary language:** C++ (90%+)
- **Secondary:** Python

Compiled and tested mainly with **GNU G++** (C++17 / C++20).
